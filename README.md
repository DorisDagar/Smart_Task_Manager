<h1 align="center">📝 Smart Task Manager</h1>

<p align="center">
A full-stack web application for organizing daily and long-term tasks with secure authentication and efficient task management.
</p>

<p align="center">
Built with FastAPI • PostgreSQL • SQLAlchemy • HTML • CSS • JavaScript
</p>

---

## 📖 Overview

Smart Task Manager is a productivity-focused web application that allows users to manage daily and long-term tasks through a clean and intuitive interface. It includes secure authentication, task tracking, and persistent data storage, making it easy to organize work efficiently.

---

## ✨ Features

- 🔐 Secure User Registration & Login
- 👤 Session-Based Authentication
- 📅 Daily Task Management
- 🎯 Long-Term Task Management
- ✅ Task Completion Tracking
- 📜 Task History
- 🗄️ PostgreSQL Database Integration
- ⚡ RESTful APIs using FastAPI

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/DorisDagar/Smart_Task_Manager.git
```

### 2️⃣ Navigate to the project directory

```bash
cd Smart_Task_Manager
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
uvicorn main:app --reload
```

The application will be available at:

```text
http://127.0.0.1:8000
```
